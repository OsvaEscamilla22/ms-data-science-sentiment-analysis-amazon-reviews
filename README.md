# MS Data Science Sentiment Analysis – Amazon Reviews
Master’s Data Science project on multiclass sentiment classification of e-commerce reviews with Python and scikit-learn.

## Objective
This project focuses on multiclass sentiment classification of e-commerce reviews using machine learning. The objective was to analyze customer reviews from Amazon mobile phone products and classify sentiment into positive, negative, and neutral categories based on review ratings.

## Dataset
The dataset used in this project contains Amazon mobile phone reviews, including review text and rating values. For the analysis, review text was used as the main predictor variable and sentiment labels were engineered from rating scores.

## Methodology
The workflow included data cleaning, target label generation, train-test split, TF-IDF vectorization, and model comparison. The models evaluated were Decision Tree, Random Forest, and Support Vector Machine.

## Tools
Python, pandas, NumPy, scikit-learn, matplotlib, seaborn, and Google Colab.

## Key Insights
The Support Vector Machine model achieved the best overall performance for multiclass sentiment classification, outperforming Decision Tree and Random Forest on accuracy and class balance.

## Results
This project demonstrates how NLP and machine learning can be applied to customer review analysis in e-commerce environments, transforming unstructured text into actionable insights.
